# Advent of Code 2025

TypeScript solutions for [Advent of Code 2025](https://adventofcode.com/2025).

## Theme

> The Elves have discovered project management! According to their resource planning, none of them have any time left to decorate the North Pole. Help them finish decorating by December 12th.

## Setup

```bash
npm install
```

## Usage

### Create a new day

```bash
npm run new 1
```

Creates `src/day01/` with:
- `solution.ts` — solution template
- `solution.test.ts` — test file with example input
- `input.txt` — empty file for puzzle input

### Run a solution

```bash
npm run day src/day01/solution.ts
```

### Run tests

```bash
# All tests
npm test

# Single day
npm run test:day src/day01/solution.test.ts
```
